# JPplusBuildings

Welcome to 🇯🇵🏠 **JP+ Buildings** 🏠🇯🇵

## What is this set?

* Japanese themed buildings
* Passenger Service requirement for buildings greater than two storeys
* Larger cities and wider roads allow for even taller buildings
